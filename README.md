Este projeto é um **card de receita responsivo** desenvolvido com **HTML e CSS**.
Ele apresenta uma receita, com imagem, título, descrição e um botão de chamada para ação.

## 🚀 Tecnologias Utilizadas

* **HTML5**
* **CSS3**
